Getting Started with Kid Tasker

A quick guide to setting up your family's weekly task tracking.

What is Kid Tasker?

Kid Tasker helps families build positive habits by turning weekly chores and responsibilities into a printed checklist. Each week, you print a worksheet for each child. The child checks off tasks as they complete them, and at the end of the week a parent reviews the worksheet by scanning it back in. The system reads the checkboxes automatically and tracks progress over time through analytics.

Initial Setup

1
Create an account. Sign up with your email and password. This creates your parent account.
2
Create your family. Give your family a name. If a second parent wants to join, go to Settings and send them an invite.
3
Add your children. In Settings, add each child with their name and birthday. A set of default tasks will be pre-loaded for each child.
4
Customize tasks. From the Dashboard, tap a child's name to open their Checklist. Here you can add, remove, reorder, and categorize tasks. Each task can be assigned to specific days of the week.

Weekly Workflow

1
Print the worksheet. Each week, go to the child's Checklist page and tap "Generate & Print." This creates a PDF with all tasks, checkboxes for each day, and a QR code that identifies the worksheet. Print it on standard letter paper.
2
Kids check off tasks. Throughout the week, your child marks the left checkbox for each task they complete. At the end of the week, a parent marks the right checkbox to confirm completion.
3
Scan and submit. Go to the Submit page and upload a photo or flatbed scan of the completed worksheet. The system reads the QR code to identify the worksheet, then analyzes the checkboxes automatically. The image can be in any orientation.
4
Review results. After scanning, you'll see a results table showing what the system detected for both child-reported and parent-reported checkboxes. You can correct any misreads before saving.
If your child marked tasks in blank rows that weren't pre-printed, the system will detect those too. You'll be prompted to name these "ad-hoc" tasks during review, and they can be added to future worksheets.

Analytics

The Analytics page shows completion trends over time, broken down by category and day of week. The "Completion Rate Over Time" chart shows both child-reported and parent-reported results side by side, so you can see how closely they align. Category and day-of-week charts currently report parent-confirmed data only.

Exporting Data

In Settings, each child has a "Download Data" button that exports all scored worksheet data as a CSV file. The export includes every task grouped by category, with columns for each date showing both child-reported and parent-reported results. The child's name is not included in the file for privacy.

Tips for Best Results

Printing: Print at actual size (100%) on standard letter paper. The worksheet has registration marks set inward from the edges, so you should not need "Fit to Page."

Marking checkboxes: Use a pen, pencil, or marker. Any clear mark inside the box will be detected. The printed checkbox borders are intentionally very light so the scanner ignores them.

Scanning: A flatbed scanner gives the best results. Phone photos also work. The image can be in any orientation — the system will auto-correct rotation.

Multiple parents: Both parents have full access. Invite a second parent from Settings and they can generate worksheets, scan, and review results too.
